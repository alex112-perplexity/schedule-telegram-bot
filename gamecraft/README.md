# GameCraft

Мощная платформа для создания игр на основе web-технологий.

## Описание
GameCraft - это полнофункциональная платформа для разработки, хостинга и управления веб-играми.

## Технологический стек
- Frontend: React, HTML5, CSS3, JavaScript
- Backend: Node.js, Express
- Database: PostgreSQL

## Установка
```bash
cd gamecraft
npm install
npm start
```
